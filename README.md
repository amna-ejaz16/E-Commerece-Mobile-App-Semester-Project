🛒 E-Commerce Mobile App 
A native Android e-commerce application UI developed as a semester project. Built with Kotlin and Jetpack Compose, this project showcases a modern, responsive user interface following the MVVM architecture.

✨ Key Features
Authentication Screens: Form layouts for login, sign-up, and user profiles (UI only).

Product Catalog: A clean interface to browse item categories and view specific product.

Instant Checkout Flow: A simulated shopping journey where adding an item to the cart takes the user straight to checkout, ending with a "Continue Shopping" option.

🛠️ Tech Stack
Language: Kotlin

UI Framework: Jetpack Compose (Material 3)

Architecture: MVVM (Model-View-ViewModel)
